# 南京李志

本项目使用 `APlayer` 作为音频播放器，`DPlayer` 作为视频播放器，通过 `jsdelivr` 实现 CDN ，国内也可高速访问。因完全使用 github 做资源存储，没有自己搭建服务器或使用云盘，所以理论上资源永久有效，资源实际存储在 [lizhi-resource](https://github.com/GoldSubmarine/lizhi-resource) 中。
